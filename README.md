# unrd.ru

Простой лендинг-заглушка для хаба проектов unrd.ru с RAG-чатботом.

## Настройка API ключа

Приложение использует Google Gemini API. Ключ должен быть в файле `.env` в корне проекта.

### Именование ключа
В зависимости от вашего сборщика (Vite, Webpack и т.д.), назовите переменную по-разному:

1. **Vite** (самый частый случай):
   ```
   VITE_API_KEY=ваш_ключ
   ```

2. **Обычный Node/Webpack**:
   ```
   API_KEY=ваш_ключ
   ```

3. **Create React App**:
   ```
   REACT_APP_API_KEY=ваш_ключ
   ```

**Важно:** После изменения `.env` файла обязательно перезапустите локальный сервер (`npm run dev` или аналогичную команду).

## Деплой и структура

Обязательно для деплоя:
- `index.html`
- `index.tsx`
- `App.tsx`
- `.env` (не забудьте добавить переменную окружения в панели управления хостинга)
